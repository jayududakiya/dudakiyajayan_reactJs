What Can CSS Do?

CSS can be used to turn an HTML document into a professional, polished design. Here are a few of the things you can accomplish wish CSS: Create a flexible grid for designing fully responsive websites that render beautifully on any device. Style everything from typography, to tables, to forms, and more. Position elements on a web page relative to one another using properties such as float, guide, position, overflow, flex, and box-sizing. Add background images to any element. Create shapes, interactions, and animations. These concepts and techniques go beyond the scope of this introductory but the following resources will help you tackle these more advanced topics.
